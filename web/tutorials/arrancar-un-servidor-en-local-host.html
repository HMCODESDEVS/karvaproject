Arranca un servidor en LocalHost


Para abrir/arrancar un servidor en LocalHost debes :

1. Descarga el servidor de FiveM https://runtime.fivem.net/artifacts/fivem/build_server_windows/master/ 2. Descomprime todo en una carpeta, por ejemplo: C:\fivemserver 3. Crea un archivo llamado server.cfg y pega esto: sv_hostname "Servidor Local" sv_maxclients 8 endpoint_add_tcp "0.0.0.0:30120" endpoint_add_udp "0.0.0.0:30120" start mapmanager start chat start spawnmanager start sessionmanager start basic-gamemode start hardcap 4. Crea start.bat y pega: FXServer.exe +exec server.cfg 5. Ejecuta start.bat. 6. En FiveM abre la consola (F8) y escribe: connect localhost
