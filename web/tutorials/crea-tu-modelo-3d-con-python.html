¿Cómo es esto posible? Esto es posible gracias a que Blender tiene la opción de crear un código en python y convertir el python a un Modelo 3D dentro del mismo programa.


¿A qué me puede ayudar esto? Esto te puede ayudar a agilizar el proceso de creación de modelos 3D con IA de manera fácil y sin comerte el coco Para crear tus modelos 3D usando Python :

Primero abre Blender y tu navegador favorito con la IA Claude.


Usa este prompt : "Actúa como un creador de código python para crear modelos 3D en Blender"


Cuando te conteste , di : "Ahora crea en el lenguaje python , un modelo 3D que [lo que quieras crear]"


Cuando te genere el código completo , ve a Blender en la barra de arriba , navega hasta encontrar la palabra "Script" , dale click y copia en la consola el código , luego vete a Layout.


Listo , ya tienes tu modelo 3D creado con IA
